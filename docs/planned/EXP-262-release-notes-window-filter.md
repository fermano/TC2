# EXP-262 - release-notes --since/--until window filter (placeholder)

Linear: https://linear.app/experttc/issue/EXP-262/add-since-until-window-filter-to-release-notes-generation
Category: Feature | Priority: Medium

## Problem
Release notes are generated over the full history each run, so there is no way to scope notes to a single release window.

## Planned change (placeholder)
Add --since and optional --until flags (inclusive, UTC) to the release-notes generation tooling. No flags keeps current full-history behavior. No production logic changed in this placeholder PR.
